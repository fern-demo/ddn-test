## Notes

- **Caption**: Each table has a caption that appears above it
- **Header Rows**: Use `headerRows={1}` to make the first row a header
- **Header Columns**: Use `headerColumns={1}` to make the first column headers
- **Row Span**: Use `[r3]` to span 3 rows, followed by `_` placeholders
- **Column Span**: Use `[c2]` to span 2 columns, followed by `_` placeholders
- **Combined Spans**: Use `[r2c2]` to span both rows and columns
- **Markdown**: Cells support markdown including links, code, bold, italic, etc.
- **Line Breaks**: For simple line breaks within cells without creating a new paragraph, use `\` or `<br/>` tags.
- **Paragraphs**: For multi-paragraph content within a cell, separate paragraphs with blank lines.
- **Lists**: Use normal Markdown list syntax within cells.
